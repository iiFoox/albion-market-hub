## Quick Start

1. Use the current install/update wizard: `../docs/en/20-INSTALL-UPDATE-WIZARD.md` or `../docs/pt-br/20-WIZARD-DE-INSTALACAO-E-UPDATE.md`
2. Open a chat with your AI assistant
3. Paste the activation prompt from `../docs/en/00-ACTIVATION-PROMPT.md` or `../docs/pt-br/00-ACTIVATION-PROMPT.md`
4. Use the operator map: `../docs/en/22-OPERATOR-EXPERIENCE-MAP.md` or `../docs/pt-br/22-MAPA-DE-EXPERIENCIA-DO-OPERADOR.md`
5. Start coding!

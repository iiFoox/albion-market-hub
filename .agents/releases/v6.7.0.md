# HEPHAESTUS v6.7.0 — Real Project Adapter

> Release type: Minor
> Date: 2026-04-27
> Scope: Project-local adapter contract for real repositories
> Changelog: [CHANGELOG.md](CHANGELOG.md)

---

## Summary

v6.7.0 gives HEPHAESTUS a real project adapter contract through `.agents/config/project.yaml`.

The framework can now understand project-local stack, commands, source paths, test paths, docs paths, protected paths, execution mode, and quality gates before planning controlled work in a real repository.

## Added

- `.agents/protocols/real-project-adapter-protocol.md`
  - defines required adapter fields, readiness rules, safety rules, and output contract.
- `.agents/workflows/real-project-adapter-workflow.md`
  - defines the validation flow for real local project adapters.
- `.agents/reports/adapters/README.md`
  - explains adapter reports and token policy.
- `.agents/reports/adapters/real-project-adapter-status-template.md`
  - compact status template for adapter reviews.
- `.agents/tools/check-real-project-adapter.ps1`
  - validates project adapter files, config, loading group, docs, and translation map.
- `.agents/docs/en/13-REAL-PROJECT-ADAPTER.md`
- `.agents/docs/pt-br/13-ADAPTADOR-DE-PROJETO-REAL.md`

## Changed

- Expanded `.agents/config/project.yaml` into a real adapter template.
- Added `real_project_adapter` to active framework execution config.
- Added the `adapter` conditional group to Smart Loading.
- Added Real Project Adapter validation to the pre-release gate.
- Updated the integrity manifest to expect `17` protocols and `11` workflows.
- Updated EN/PT-BR documentation map and complete references.
- Updated roadmap and changelog for real-project adaptation.

## Preserved

- Default adapter execution mode is `analysis_only`.
- Real-project work still requires explicit plan and approval before changes.
- Protected paths are explicit and conservative by default.
- Adapter context is conditional and does not load during normal LITE tasks.
- v6.6 Project Discovery and v6.5 Documentation Enforcement remain active.

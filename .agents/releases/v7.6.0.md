# HEPHAESTUS v7.6.0 — Install and Update Wizard

> Release Date: 2026-04-28
> Package: `HEPHAESTUS-Framework-v7.6.0.zip`
> Scope: Operator-guided installation and update documentation over existing safe tooling

---

## Summary

v7.6.0 formalizes a step-by-step install/update wizard for operators. The framework already had safe dry-run installer and updater tooling; this release adds the missing guided layer so users can install, update, compare, preserve local project state, and validate results without guessing which command to use.

## Added

- `.agents/protocols/install-update-wizard-protocol.md`
  - defines the decision flow for install versus update;
  - keeps dry-run-first behavior mandatory;
  - preserves project-local config by default;
  - limits loaded context for token economy.
- `.agents/tools/check-install-update-wizard.ps1`
  - verifies the wizard protocol, guides, config, loading group, and gate integration.
- `.agents/docs/en/20-INSTALL-UPDATE-WIZARD.md`
  - English operator guide for install/update.
- `.agents/docs/pt-br/20-WIZARD-DE-INSTALACAO-E-UPDATE.md`
  - Portuguese operator guide for install/update.

## Changed

- Updated active framework metadata to `7.6.0`.
- Added `install_update_wizard` to active framework config.
- Added `install-update-wizard` Smart Loading group.
- Added the wizard checker to the pre-release gate.
- Updated Getting Started docs to distinguish install/update from project onboarding.
- Updated bilingual documentation indexes, complete references, translation map, roadmap, changelog, and package checklist.
- Fixed release evidence report generation so `framework_version` follows the requested/current release instead of a stale hardcoded value.

## Safety and Compatibility

- No installer behavior was made more permissive.
- Dry-run remains the default.
- Update still preserves project-local config and session files by default.
- No network access is required.
- Normal framework use does not load the wizard docs unless install/update context is relevant.

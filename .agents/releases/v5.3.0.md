# HEPHAESTUS v5.3.0 — Smart Loading Manifest

> Release type: Minor
> Date: 2026-04-23
> Scope: Auditable Smart Loading tiers, token estimation, and loading reports
> Changelog: [CHANGELOG.md](CHANGELOG.md)

---

## Summary

v5.3.0 makes Smart Loading measurable. Instead of relying only on prose in the protocol, the framework now has an explicit loading tier manifest and a local estimator for file count, bytes, and approximate token cost.

This improves context discipline without increasing the default loading footprint.

## Added

- `.agents/config/loading-tiers.yaml`
  - defines core files;
  - defines LITE, STANDARD, DEEP, and CRITICAL tier membership;
  - defines conditional groups such as `ui`, `platform`, `flutter`, `docs`, `release`, `database`, `api`, and `resilience`.
- `.agents/tools/estimate-loading.ps1`
  - estimates file count, bytes, and approximate tokens for a selected tier;
  - supports conditional groups;
  - can show per-file details with `-ShowFiles`.
- `.agents/reports/loading/README.md`
  - documents how loading estimates should be generated and interpreted.

## Changed

- Updated `smart-loading-protocol.md` to point to `.agents/config/loading-tiers.yaml` as the auditable source of truth.
- Updated framework metadata to `5.3.0`.
- Updated the integrity manifest to require:
  - `loading-tiers.yaml`;
  - `estimate-loading.ps1`;
  - loading report documentation;
  - this release note.

## Preserved

- No broad technology catalog was added back to active config.
- No tier was made mandatory beyond the existing Smart Loading model.
- Conditional groups remain opt-in based on task relevance.
- Existing agent behavior and workflow routing remain unchanged.

## Usage

Estimate a base tier:

```powershell
powershell -ExecutionPolicy Bypass -File .agents/tools/estimate-loading.ps1 -Tier lite
```

Estimate a tier with conditional groups:

```powershell
powershell -ExecutionPolicy Bypass -File .agents/tools/estimate-loading.ps1 -Tier standard -IncludeGroups ui,flutter
```

Show per-file details:

```powershell
powershell -ExecutionPolicy Bypass -File .agents/tools/estimate-loading.ps1 -Tier deep -IncludeGroups platform -ShowFiles
```

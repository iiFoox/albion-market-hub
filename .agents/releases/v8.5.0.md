# HEPHAESTUS v8.5.0 — Operator Runbook and Recovery Guide

> Release Date: 2026-04-29
> Package: `HEPHAESTUS-Framework-v8.5.0.zip`
> Status: Completed

---

## Summary

v8.5.0 adds an operator runbook and recovery guide so daily operation, release preparation, and failure recovery have a compact, validated path.

## What Changed

- Added the Operator Runbook and Recovery Protocol.
- Added EN/PT-BR runbook guides.
- Added `check-operator-runbook.ps1`.
- Integrated runbook validation into `doctor` and the pre-release gate.
- Added a Smart Loading conditional group for runbook recovery context.
- Updated the stability baseline to include the runbook capability.

## Why It Matters

The framework already has automation, gates, packaging, evidence, bootstrap, and stability checks. v8.5.0 improves operator confidence by making the next action clear when validation, release, package, install, or update operations fail.

## Compatibility

- No breaking changes.
- No new mutation behavior.
- No new broad always-loaded content.
- Install and update remain dry-run first unless Apply is explicitly approved.

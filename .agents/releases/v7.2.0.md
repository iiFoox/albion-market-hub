## Purpose

v7.2.0 adds an isolated end-to-end real-project Apply scenario.

The scenario proves that HEPHAESTUS can run an adapter quality gate, generate DryRun, perform controlled Apply with approval token, create backup evidence, audit the execution, run quality gates again, and execute restore against the latest backup artifact.

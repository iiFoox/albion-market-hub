# HEPHAESTUS v8.3.0 — Project Bootstrap Assistant

> Release Date: 2026-04-29
> Package: `HEPHAESTUS-Framework-v8.3.0.zip`

## Summary

v8.3.0 adds a Project Bootstrap Assistant that connects first install to Project Discovery and Real Project Adapter readiness.

## Added

- `.agents/tools/run-project-bootstrap.ps1`
- `.agents/tools/check-project-bootstrap.ps1`
- `.agents/protocols/project-bootstrap-assistant-protocol.md`
- `.agents/reports/adapters/project-bootstrap-template.md`
- `.agents/docs/en/27-PROJECT-BOOTSTRAP-ASSISTANT.md`
- `.agents/docs/pt-br/27-ASSISTENTE-DE-BOOTSTRAP-DE-PROJETO.md`

## Changed

- Added `bootstrap` to `.agents/tools/hephaestus.ps1`.
- Added `project_bootstrap_assistant` to active framework config.
- Added `project-bootstrap` to Smart Loading conditional groups.
- Integrated project bootstrap validation into the pre-release gate and doctor.
- Updated manifest, translation map, roadmap, changelog, and complete references.

## Safety and Compatibility

- Bootstrap is analysis-only.
- It does not overwrite `.agents/config/project.yaml`.
- It does not enable commands automatically.
- It writes optional adapter drafts under reports for operator review.
- Product story and use cases remain owned by Project Discovery.

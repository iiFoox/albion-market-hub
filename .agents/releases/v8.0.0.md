# HEPHAESTUS v8.0.0 — Unified Operator CLI and Doctor

> Release Date: 2026-04-28

## Summary

v8.0.0 adds a unified operator CLI at `.agents/tools/hephaestus.ps1`. It centralizes common framework operations without replacing or weakening the existing scripts.

- `.agents/tools/hephaestus.ps1`

- `.agents/docs/en/24-UNIFIED-OPERATOR-CLI.md`
- `.agents/docs/pt-br/24-CLI-UNIFICADO-DO-OPERADOR.md`

## Safety and Compatibility

- Install and update remain dry-run unless `-Apply` is passed.
- Update keeps backup behavior by default.
- The CLI does not require network access.
- Existing scripts remain available and unchanged as direct entry points.

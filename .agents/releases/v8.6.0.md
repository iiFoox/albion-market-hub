# HEPHAESTUS v8.6.0 — Guided Installer and Repository Onboarding

> Release Date: 2026-04-29
> Package: `HEPHAESTUS-Framework-v8.6.0.zip`
> Status: Completed

---

## Summary

v8.6.0 upgrades first install into a guided repository onboarding flow. It records whether the target project is local, GitHub, GitLab, Bitbucket, other remote, existing repository, or no repository yet, then creates a first-call handoff that the operator can paste into the AI coding assistant.

## What Changed

- Added guided repository mode prompts to the Windows installer.
- Added repository onboarding parameters to the PowerShell launcher and unified CLI install action.
- Added target project state detection for new, empty, existing, and existing Git projects.
- Added first-call handoff files after Apply install.
- Added repository setup report generation.
- Added checker, protocol, and EN/PT-BR guides.

## Safety

- No `git init`, remote add, branch rewrite, commit, or push is executed automatically.
- Existing project repositories are preserved.
- Git alignment happens organically in the first call and bootstrap flow.

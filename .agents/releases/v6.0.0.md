# HEPHAESTUS v6.0.0 — Framework Kernel Stabilization

> Release type: Major
> Date: 2026-04-27
> Scope: Stable kernel contract, compatibility policy, migration guide, and host matrix
> Changelog: [CHANGELOG.md](CHANGELOG.md)

---

## Summary

v6.0.0 declares the hardened 5.x framework as a stable kernel.

This release formalizes what must remain compatible across future 6.x versions: directory contracts, required entry points, release validation, package verification, documentation parity, telemetry validation, and loading estimates.

## Added

- `.agents/kernel/KERNEL.md`
  - defines the stable kernel surface;
  - freezes core directory and entry point contracts;
  - defines what minor and major releases may change.
- `.agents/kernel/COMPATIBILITY.md`
  - defines patch, minor, and major compatibility rules;
  - defines deprecation expectations and host requirements.
- `.agents/kernel/MIGRATION-v5-to-v6.md`
  - documents how hardened 5.x installs move to 6.0.0.
- `.agents/kernel/HOST-MATRIX.md`
  - defines supported and compatible host environments.
- `.agents/tools/check-kernel-contract.ps1`
  - validates kernel directories, kernel files, manifest registration, and kernel doc versions.

## Changed

- Updated framework metadata to `6.0.0`.
- Added kernel governance paths to active config.
- Added kernel files and validator to the integrity manifest.
- Added kernel contract validation to `.agents/tools/pre-release-gate.ps1`.
- Updated EN/PT-BR complete references with the v6.0.0 release row.
- Updated package checklist commands for v6.0.0.

## Preserved

- No normal workflow now requires loading the whole framework.
- No network-dependent validation was introduced.
- Existing Smart Loading tiers remain intact.
- Existing routing fixtures remain intact.
- Existing bilingual documentation layout remains intact.

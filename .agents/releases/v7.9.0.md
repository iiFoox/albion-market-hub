# HEPHAESTUS v7.9.0 — Real Inter-Agent Communication Bus

> Release Date: 2026-04-28
> Package: `HEPHAESTUS-Framework-v7.9.0.zip`
> Scope: Compact auditable communication bus for handoffs, consultations, conflicts, decisions, alerts, and corrections

---

## Summary

v7.9.0 adds a real inter-agent communication bus contract. The framework already had collaboration and handoff protocols; this release adds an explicit bus config, message proof runner, validator, reports, documentation, telemetry support, and release evidence integration.

## Added

- `.agents/config/communication-bus.yaml`
  - defines required message fields, allowed message types, priorities, statuses, and compactness rules.
- `.agents/protocols/inter-agent-communication-bus-protocol.md`
  - defines when and how bus messages are created.
- `.agents/tools/run-communication-bus-proof.ps1`
  - generates compact sample evidence for handoff, consultation, conflict, and decision messages.
- `.agents/tools/check-communication-bus.ps1`
  - validates bus config, protocol, reports, telemetry schema, loading group, and gate wiring.
- `.agents/reports/communication/`
  - stores communication bus templates and generated evidence.
- `.agents/docs/en/23-INTER-AGENT-COMMUNICATION-BUS.md`
- `.agents/docs/pt-br/23-BARRAMENTO-DE-COMUNICACAO-ENTRE-AGENTES.md`

## Changed

- Updated active framework metadata to `7.9.0`.
- Added `inter_agent_communication_bus` to active framework config.
- Added `communication-bus` Smart Loading group.
- Added `communication-bus` telemetry log type and canonical communication events.
- Added communication bus checker and runner to the pre-release gate.
- Added communication bus evidence to the release evidence bundle.
- Updated bilingual docs indexes, complete references, translation map, roadmap, changelog, and package checklist.

## Safety and Compatibility

- No full agent transcripts are required.
- Messages are compact summaries with references to artifacts.
- Normal sessions load the bus only when coordination/audit context is relevant.
- No project mutation behavior changed.
- No network access is required.

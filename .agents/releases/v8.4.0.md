## Summary

v8.4.0 adds a stability baseline and regression sentinel so future framework evolution can detect accidental breaks in critical operational wiring.

- Added `stability_baseline_regression_sentinel` to active framework config.
- Added `stability-baseline` to Smart Loading conditional groups.
- Integrated stability baseline validation into doctor and the pre-release gate.
- Added stability baseline evidence to the release evidence bundle.
- Updated manifest, translation map, roadmap, changelog, and complete references.

## Safety and Compatibility

- The sentinel only reads framework files.
- It does not execute project commands.
- It does not mutate project files.
- It validates critical wiring without replacing feature-specific checks.

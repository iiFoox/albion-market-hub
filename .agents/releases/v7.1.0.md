## Purpose

v7.1.0 adds a safe quality gate runner for real projects.

The framework can now parse adapter commands from `.agents/config/project.yaml`, execute only commands marked `allowed: true`, deny destructive command patterns, and write a compact command-quality report.

## Safety Model

- Commands are sourced only from `.agents/config/project.yaml`.
- Commands must be explicitly allowed.
- Destructive patterns are denied before execution.
- Default empty or disabled commands remain no-op.
- Reports stay compact to preserve token economy.

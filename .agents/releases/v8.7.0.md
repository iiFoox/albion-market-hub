# HEPHAESTUS v8.7.0 — Practical First Project Walkthrough

> Release Date: 2026-04-30
> Package: `HEPHAESTUS-Framework-v8.7.0.zip`
> Status: Completed

---

## Summary

v8.7.0 adds a practical first project walkthrough so operators can confidently use HEPHAESTUS after installation in both new and existing projects.

## What Changed

- Added EN/PT-BR first project walkthrough guides.
- Added a protocol and checker for first project usage.
- Covered new project and existing project paths.
- Made first-call, bootstrap, adapter draft, Project Discovery, repository strategy, protected paths, allowed commands, and DryRun-first readiness explicit.

## Why It Matters

The framework is powerful, but first use must be obvious. This release turns the first project experience into a concrete checklist that supports real IDE/AI assistant usage.

## Quick Start (TL;DR)

1. Copy `.agents/` folder into your project root
2. Use the **Install and Update Wizard** when copying or updating the framework (see `20-INSTALL-UPDATE-WIZARD.md`)
3. Run project onboarding (see `01-GETTING-STARTED.md`)
4. Start coding — the framework handles the rest

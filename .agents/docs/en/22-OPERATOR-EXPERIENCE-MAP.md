## Recommended First-Use Path

1. Install or update with `20-INSTALL-UPDATE-WIZARD.md`.
2. Activate with `00-ACTIVATION-PROMPT.md`.
3. Read `01-GETTING-STARTED.md`.
4. If starting a new product, run `12-PROJECT-DISCOVERY.md` before implementation.
5. If working in a real repo, configure `13-REAL-PROJECT-ADAPTER.md`.
6. Execute using `14-REAL-PROJECT-EXECUTION.md`.
7. Validate with `15-COMMAND-QUALITY-GATES.md`.
8. Package with `18-RELEASE-EVIDENCE-BUNDLE.md`.

## Low-Token Rule

Open this map first, then open only the guide that matches the operator's intent. Do not load all documentation unless doing a full documentation review.

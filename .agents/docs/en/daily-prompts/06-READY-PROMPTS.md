# 06 — Ready Prompts

Copy and adapt these prompts.

---

## Onboard Project

```text
Initialize HEPHAESTUS for this project and produce an onboarding summary before making changes.
```

## Implement Feature

```text
Use HEPHAESTUS STANDARD workflow to implement:

[feature]

Include planning, implementation, validation, and documentation where relevant.
```

## Investigate Bug

```text
Use HEPHAESTUS error-correction workflow to investigate:

[bug]

Find root cause, propose the fix, implement it, and validate the result.
```

## Review Only

```text
Use HEPHAESTUS review-only workflow.

Review this code/change for correctness, regressions, security, maintainability, and missing tests. Do not edit files unless I explicitly approve.
```

## UI/UX Work

```text
Use HEPHAESTUS UI/UX workflow for:

[screen/component/change]

Consider responsiveness, accessibility, visual hierarchy, empty states, and interaction states.
```

## Close Session

```text
Close this HEPHAESTUS session and produce a handoff note for the next chat.
```

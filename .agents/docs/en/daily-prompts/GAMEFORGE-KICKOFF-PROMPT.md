# GameForge Kickoff Prompt

> Use when: Starting a game/tool project that should use HEPHAESTUS as a disciplined software delivery framework.

---

```text
Activate HEPHAESTUS for this GameForge-style project.

Read .agents/AGENTS.md and the relevant documentation under .agents/docs/en/.

Then inspect the project and produce:

1. project summary;
2. detected stack and runtime;
3. gameplay/tooling surface;
4. relevant agents;
5. risks and unknowns;
6. recommended first milestone;
7. validation strategy;
8. documentation and delivery plan.

If this includes UI, game loops, canvas, animation, assets, or 3D, include UI/UX Specialist and Platform Guardian in the triage.

Do not edit files until the first milestone is clear.
```

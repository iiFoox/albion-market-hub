# 03 — Daily Use

> Use when: You are working normally with HEPHAESTUS.

---

## For a Simple Fix

```text
Use the HEPHAESTUS quick-fix workflow for this task:

[describe the issue]

Classify the task, activate only the relevant agents, implement the fix, and validate the result.
```

## For a Feature

```text
Use the HEPHAESTUS standard workflow for this feature:

[describe the feature]

Include planning, implementation, validation, and documentation updates where relevant.
```

## For Architecture or Risky Work

```text
Use the HEPHAESTUS advanced workflow.

Before editing files, analyze architecture, risks, affected modules, migration concerns, testing strategy, and rollback considerations.

Task:
[describe the task]
```

## For Production-Critical Work

```text
Use the HEPHAESTUS critical workflow.

Treat this as production-sensitive. Include security, regression, compatibility, testing, documentation, and delivery gates.

Task:
[describe the task]
```

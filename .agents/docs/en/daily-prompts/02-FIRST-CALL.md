# 02 — First Call

> Use when: You are activating HEPHAESTUS in a project for the first time.

---

## Prompt

```text
Activate HEPHAESTUS for this project.

Read the operational contract in .agents/AGENTS.md.
Then inspect the project structure and summarize:

1. detected stack and platforms;
2. active profiles that should apply;
3. relevant agents for this project;
4. likely risks or missing context;
5. recommended first workflow;
6. what you need from me before making changes.

Keep framework internals in English, but communicate with me in my preferred language.
```

## Expected Output

The assistant should not start changing files immediately. It should first produce an onboarding summary and propose the next step.

# 04 — Session Closing

> Use when: You are ending a work session.

---

## Prompt

```text
Close this HEPHAESTUS session.

Summarize:
1. what was requested;
2. what was changed;
3. files touched;
4. validation performed;
5. remaining risks;
6. next recommended steps;
7. memory entries or lessons that should be preserved;
8. whether a commit, changelog, or release note is needed.

Create a concise handoff note that can be used to resume in a new chat.
```

## Expected Output

The assistant should produce a compact session handoff and identify whether delivery actions are still needed.

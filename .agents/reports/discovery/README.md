# Discovery Reports

> Framework Version: 7.5.0

This directory stores project discovery and requirements maturation outputs.

Discovery reports are living documents. They should be updated when the operator changes product direction, adds business rules, chooses a technical path, changes cost assumptions, or confirms implementation readiness.

## Files

| File | Purpose |
|---|---|
| `project-discovery-template.md` | Full project discovery report template |
| `decision-log-template.md` | Compact record for direction-changing decisions |

## Token Policy

Do not load every discovery report by default. Load the latest summary or the specific report relevant to the current project, then update only the changed sections.

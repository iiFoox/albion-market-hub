# Developer Execution Reports

> Framework Version: 7.5.0
> Purpose: Store compact reports from controlled developer execution simulations.

---

Developer execution reports are generated by:

```powershell
powershell -ExecutionPolicy Bypass -File .agents/tools/run-developer-execution-simulation.ps1 -WriteReport
```

The execution harness creates a disposable mini project, implements a small feature, runs local structural tests, validates the installed framework, writes a compact delivery report, and removes the workspace by default.

# Simulation Reports

> Framework Version: 7.5.0
> Purpose: Store generated summaries from isolated framework simulations.

---

Simulation reports are generated by:

```powershell
powershell -ExecutionPolicy Bypass -File .agents/tools/run-framework-simulation.ps1 -WriteReport
```

The harness creates an isolated temporary workspace, installs the framework there, runs validation and routing checks, writes a compact report, and removes the temporary workspace by default.

Use `-KeepWorkspace` only when debugging a simulation manually.
